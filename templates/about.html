About

This is an automated life-archive system that transforms Claude conversation exports into a structured, timestamped public record.

Design Philosophy

  • Exhaust, not effort: Raw material from normal AI conversations, processed into structured archive without dedicated "writing time"
  • Timestamped provenance: Every entry carries verifiable timestamps as proof-of-thinking
  • Raw over polished: Preserves the texture of thinking-in-progress, avoiding corporate blog voice
  • Zero daily maintenance: System runs on scheduled triggers after initial setup

Content Types

  • Prediction: Explicit claims about future events with timelines
  • Insight: Novel connections or pattern recognition
  • Milestone: Project completions, infrastructure changes, significant decisions
  • Reflection: Personal processing, emotional content, life context
  • Technical: Build logs, configuration details, troubleshooting

How It Works

  1. Export conversations from claude.ai
  2. Ingest and deduplicate exports
  3. Extract themes, summaries, and classifications using LLM
  4. Compile weekly digests and theme threads
  5. Generate static site and deploy

Technical Stack

Built with Rust. Static site hosted on Internet Computer Protocol. Source available at GitHub.
